## Dev tools

It needs to be executed at the root of the project and have `Leiningen` installed.

### Check

It checks linguistically and syntaxically if the code is correct.

``` shell
make check
```

### Build

Build a JAR ready to distribute.

``` shell
make build
```

### Deploy

Distributed in Clojars.

``` shell
make deploy
```
